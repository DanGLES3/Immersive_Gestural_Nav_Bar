# Immersive Gestural Navigation Bar for Android 10

This is a simple Magisk module enables immersive gestural navigation bar for Android 10.

## Requirements
- Android 10 only
- Magisk 18+

## Installation
1. Flash this module.
2. Reboot.
3. Enjoy immersive gestural navbar!
